## Description
This project is a 2D side-scrolling platformer built with p5.js and p5play.  
The player controls a king character, collects diamonds, avoids fire hazards, and fights boar enemies.

Users can choose a level at the start, move through platform challenges, collect all required diamonds, and survive enemy/hazard encounters.  
The design was inspired by pixel-art action platformers with layered parallax backgrounds and arcade-style HUD feedback.

---
## Interaction Instructions
- Start the game, then choose a level:
  - Press `1` for Level 1
  - Press `2` for Level 2
- Controls:
  - `A` / `Left Arrow`: Move left
  - `D` / `Right Arrow`: Move right
  - `W` / `Up Arrow`: Jump
  - `Space`: Attack
  - `R`: Restart after death or win
- Gameplay feedback:
  - HUD shows rescued diamond count and health hearts.
  - Character flashes when damaged.
  - Audio feedback plays for jump, attack, hurt, collection, and game over.
  - Background music loops during gameplay.
